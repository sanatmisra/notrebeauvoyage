# Image Library

Use these folders to keep the invitation assets organized:

- `couple/`: older photos of Sanat and Sneha for story and relationship moments
- `trip/`: Riviera travel imagery for destinations, mood, and itinerary sections

Suggested filenames:

- `couple/01-munich-walk.jpg`
- `couple/02-anniversary-dinner.jpg`
- `trip/st-tropez-harbor.jpg`
- `trip/nice-promenade.jpg`

Guidelines:

- Prefer landscape images for the hero and destination sections.
- Keep original filenames readable and stable.
- Optimize large photos before committing when possible.
